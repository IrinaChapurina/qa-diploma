# План-тест по автоматизации тестирования возможности покупки тура в приложении на веб-сервисе.
## 1. Планирование автоматизации.
### 1.1. Тестирование формы "Оплата по карте".
Перейти на страницу "Путешествие дня", выбрать форму покупки тура "Купить", нажать на кнопку "Купить", внизу появляется форма для заполнения данных «Оплата по карте».

#### Позитивные сценарии:

 1.	Оплата путешествия дня по карте валидными данными с текущим сроком действия банковской карты:<br>
 1.1.	Заполнить поле формы «Номер карты» валидными данными (4444 4444 4444 4441);<br>
 1.2.	Заполнить поле «Месяц» валидными данными (09);<br>
 1.3.	Заполнить поле «Год» валидными данными (23);<br>
 1.4.	Заполнить поле «Владелец» на латинице в верхнем регистре (IVANOV IVAN);<br>
 1.5.	Заполнить поле «CVC/CVV» валидными данными (111);<br>
 1.6.	Нажать кнопку "Продолжить";<br>
Ожидаемый результат: отображается «отправляем запрос в банк», а затем «Успешно. Операция одобрена банком.».<br>

2. Оплата путешествия дня по карте валидными данными с текущим сроком действия банковской карты + один месяц:<br>

2.5.	Заполнить поле «CVC/CVV» валидными данными (111);<br>

## 2.	Перечень используемых инструментов с обоснованием выбора:
•	IntelliJ IDEA- среда разработки.<br>
•	Java 11 - язык программирования (так как в данной версии разрабатываются большинство проектов).<br>
•	Selenide - фреймворк для автоматизированного запуска веб-приложений по поиску веб-элементов на странице (в отличие от Selenium не требует настройки окружения, установки специального драйвера).<br>
•	Allure — фреймворк, предназначенный для создания визуально наглядной картины отчета о выполнении тестов. (наиболее распространен).<br>
•	Lombok - Плагин для минимизации строк кода за счет аннотаций позволяет оптимизировать код автотестов, добавляет в Java новые «ключевые слова» и превращает аннотации в Java-код, уменьшая усилия на разработку и обеспечивая дополнительную функциональность.<br>
•	JavaFaker - библиотека для генерации случайных тестовых данных.<br>
•	для API тестирования: Rest Assured - java-библиотека для тестирования REST API, позволяет автоматизировать тестирование get и post запросов.<br>
•	Git и GitHub - для хранения кода. Git достаточно прост и удобен для управления исходным кодом, очень распространенная система контроля версий, поэтому достаточно хорошо взаимодействует с различными ОС. GitHub специализированный веб-сервис с удобным интерфейсои, интегрирован с Git.<br>
•	Docker Desktop - это платформа контейнеризации с открытым исходным кодом. для развертывания контейнера с базой данных MySQL.<br>
